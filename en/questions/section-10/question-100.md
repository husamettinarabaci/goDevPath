### 🔹 Category: Maps  
#### ❓ Question 100: Maps and JSON encoding/decoding

Write a Go program that demonstrates how to encode a map to JSON and decode JSON back into a map.

- Create a map with some key-value pairs.
- Use the `encoding/json` package to marshal the map to a JSON string.
- Unmarshal the JSON string back into a new map.
- Print both the JSON string and the resulting map.

🔧 **Task:** Use the `json.Marshal` and `json.Unmarshal` functions to convert a map to JSON and back.
