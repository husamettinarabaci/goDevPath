#### ❓ Question 92: Adding and removing map entries

Write a Go program that demonstrates how to add and remove entries in a map.

- Declare and initialize a map with at least one key-value pair.
- Add a new key-value pair to the map.
- Remove a key-value pair from the map using the `delete` function.
- Print the map after each operation.

🔧 **Task:** Show how to add and remove entries in a Go map, and print the map after each change.
