#### âœ… Answer 8: Using escape characters in strings

This example shows how to use escape characters in Go strings. The `\t` adds a tab, and the `\n` adds a new line.

```go
package main
import "fmt"

func main() {
    fmt.Println("Hello\tGo\nNew Line!")
}
```
