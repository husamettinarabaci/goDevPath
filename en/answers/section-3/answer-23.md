#### âœ… Answer 23: Using `for` loops (basic iteration)

In Go, the `for` loop is used for iteration. Here is an example that prints numbers 1 to 5:

```go
package main
import "fmt"

func main() {
    for i := 1; i <= 5; i++ {
        fmt.Println(i)
    }
}
```
